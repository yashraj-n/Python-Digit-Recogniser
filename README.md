## Run Locally

Clone the project

```bash
  git clone https://github.com/yashraj-n/python-digit-recogniser
```

Go to the project directory

```bash
  cd python-digit-recogniser
```

Install dependencies

```bash
  pip i keras tensorflow Pillow tkinter pywin32
```

Run the Program

```bash
  python digit.py
```
